If you set general.cgi, ssc will check these environment variables:

QUERY_STRINGCGI command, intended for use with OpenBSD’s httpd server. See notes below.
SSC_CONFIGIf no configuration file is given on the command line, use this one.
SSC_ARGSPreliminary command line parameters.

If, when is run, the environment variable QUERY_STRING is set to an OpenBSD httpd server CGI value that includes the parameter html.snippet, then will nitpick that snippet only. As the name suggests, the snippet must be HTML. Certain other parameters are processed, including general.verbose and html.version.
