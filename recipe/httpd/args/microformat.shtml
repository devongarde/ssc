These switches are useful for checking andor outputting any microformat data found.

microformat.exportExport microformat data encountered in JSON format. This option will write files in the same directory as the source, with the extension .json.
microformat.verify
-M		Verify Microformats data in class and rel attributes.
microformat.version xPresume microformats version x. The following values are currently accepted:
1microformats version 1 only
2microformats version 2 only
3both microformats versions 1 and 2
